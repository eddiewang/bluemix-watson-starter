Watson Natural Language Classifier

Demo for COMM190 that uses the NLC service to analyze and categorize customer reviews from Amazon.

Instructions

This application uses the NLC service to categorize customer reviews as either: (1) Overjoyed, (2) Satisfied, (3) Ambivalent, (4) Product Complaint, (5) Process Complaint, or (6) Gibberish.

Enter in a comment that would be representative for a customer review, or select a random review, and press analyze.
